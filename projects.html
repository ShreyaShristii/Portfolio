Event Management System

Eventora is a web-based event management system built to replace fragmented manual coordination with structured, traceable workflows.

The project prioritizes understanding backend logic, data flow, and user-driven actions rather than relying heavily on abstractions or frameworks.

  • Structured event and booking data handling
  • Venue discovery and filtering logic
  • Admin-side event management workflows
  • Designed with clarity, scalability, and maintainability in mind

Tech Stack: HTML · CSS · JavaScript · PHP · MySQL
